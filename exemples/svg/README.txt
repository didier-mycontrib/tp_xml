SVG = Scalable Vector Graphic 
    = format d'images vectorielles (avec lignes , cercles , rectangles, ...)
      encodé en XML

namespace officiel : xmlns="http://www.w3.org/2000/svg"
à placer dans un fichier .svg pour une bonne visualisation 
à partir d'un navigateur

NB: le logiciel open source "inkscape" 
permet de lire et écrire des fichiers SVG sophistiqués
